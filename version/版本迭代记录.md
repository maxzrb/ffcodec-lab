## 当前版本

- 版本号：0.2.0
- 发布日期：2026-07-10
- 状态：第二阶段核心功能完成（可用 MVP，内部里程碑）
- 主要变化：
  - **参数核验模型修正**：`SourceAuthority` + `VerificationLevel` 分离，取代旧的 `status: verified` 语义
  - **字段解析层**：`src/domain/presentation/` — 纯 TypeScript，将 Config+Catalog+Rules 转为 UI 可渲染的 `ResolvedField`
  - **正式 BuilderPage**：6 个可折叠参数区域，零 FFmpeg 硬编码，所有控件由 `ResolvedField` 驱动
  - **参数解释面板**：显示说明、影响评估、来源引用和核验状态，从字段？按钮和命令 token 均可打开
  - **命令预览**：Bash/PowerShell/CMD 切换、单行/多行、复制、token 点击定位控件、两遍编码双命令显示
  - **字幕混流/烧录 UI**：内外挂字幕、容器自适应编码、基础 style 设置、与 copy 模式冲突检测
  - **预设与持久化**：localStorage 存储、CRUD 操作、JSON 导入导出、Zod 验证、schema migration、5 个内置预设
  - **测试扩充**：76 单元测试（+43 new：25 presentation + 18 presets）
  - **验收**：10 个手工命令案例全部通过，记录在 `docs/mvp-acceptance.md`
- 影响范围：新增 16 文件，修改 15 文件；原有架构和测试零破坏
- 验证情况：TypeScript 0 错误、目录审计 0 错误 0 警告、76 测试通过、生产构建成功（263KB）、10/10 验收通过
- 已知风险：5 项参数 `needsCrossVerification: true`、PresetManager UI 组件待开发、无 React 组件测试
- 迁移说明：类型系统新增 `sourceAuthority`/`verificationLevel`/`needsCrossVerification` 必填字段，所有编码器和参数数据已更新
